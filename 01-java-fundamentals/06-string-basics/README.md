# String Basics

## Overview

This module introduces the fundamental concepts of Java Strings.

You will learn what a String is, how String variables work, how concatenation is performed, and why String objects are immutable.

The goal of this module is to build a strong understanding of String behavior before learning String methods.

---

## Learning Objectives

After completing this module, you should be able to:

- Understand what a String is.
- Declare and initialize String variables.
- Concatenate Strings using the + operator.
- Understand operator precedence with Strings.
- Explain the concept of String references.
- Understand why String objects are immutable.
- Use the concat() method correctly.
