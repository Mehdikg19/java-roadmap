Notes

Key Points

A variable is a named memory location used to store data.

Every variable must have a data type.

Java is a statically typed language, so every variable must be declared before it is used.

Variables can be declared first and initialized later, or both in a single statement.

Primitive data types store simple values directly.

Use meaningful variable names and follow the camelCase naming convention.

Use System.out.println() to display output on the console.

Common Mistakes

Using String.out.println() instead of System.out.println().

Forgetting the semicolon (;) at the end of a statement.

Using the wrong data type for a value.

Choosing unclear variable names such as a, x, or temp.

Forgetting to initialize a variable before using it.

Clean Code Tips

Use descriptive and meaningful variable names.

Keep one statement per line.

Group related variables together.

Leave blank lines between logical sections of code.

Follow consistent indentation and formatting.

Write code that is easy to read and maintain.
