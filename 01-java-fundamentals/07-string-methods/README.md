## Important Notes

- String objects are **immutable**.
- Most String methods return a **new String**.
- Always assign the returned value if you want to keep the changes.
- `substring(begin, end)` starts from `begin` and stops **before** `end`.
- `equals()` compares text exactly.
- `equalsIgnoreCase()` ignores uppercase and lowercase differences.
