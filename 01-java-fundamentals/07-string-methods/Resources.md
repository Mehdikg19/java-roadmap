# Additional Practice Ideas

Try creating programs that:

- Count characters in a sentence.
- Remove extra spaces from user input.
- Convert names to uppercase.
- Convert emails to lowercase.
- Replace forbidden words.
- Extract username from an email address.
- Check if a file name ends with ".java".
- Check if a sentence starts with a specific word.
- Compare passwords ignoring uppercase/lowercase.
- Practice substring() using different beginIndex and endIndex values.

---

# Challenge

Create a program that receives a person's full name and:

- Removes extra spaces.
- Converts the name to uppercase.
- Prints the first name.
- Prints the last name.
- Prints the total number of characters.

Use:

- trim()
- substring()
- contains()
- length()
- toUpperCase()
