# Module 05 - Constants & final

## Overview

This module introduces the final keyword in Java and explains how to create constants. You will learn why constants improve code readability, maintainability, and reliability by preventing accidental modifications.

## Learning Objectives

After completing this module, you will be able to:

- Understand the purpose of the final keyword.
- Declare and use constants.
- Follow Java naming conventions for constants.
- Differentiate between variables and constants.
- Understand when a final variable can be initialized.
- Recognize compile-time errors related to final variables.
- Write cleaner and more maintainable code using constants.
