# Module 04 — Input / Output (Scanner)

## Overview

This module introduces the Scanner class, one of the most frequently used classes in Java for reading user input from the keyboard.

During this module, we learned how to read different types of input, understood the differences between Scanner methods, and solved one of the most common beginner problems: the Scanner Buffer Problem.

# Learning Objectives

After completing this module, you should be able to:

- Read different primitive data types from the keyboard.

# Key Concepts

| Method | Description |
|---------|-------------|
| next() | Reads one word |
| nextLine() | Reads an entire line |
| nextInt() | Reads an integer |
| nextDouble() | Reads a decimal number |

---

# Best Practices

- Always close the Scanner object.
- Use meaningful prompts for users.
- Use nextLine() whenever spaces are expected.
- Clear the input buffer before calling nextLine() after numeric input.
- Format console output for better readability.
