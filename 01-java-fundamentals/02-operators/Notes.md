Important Concepts

= assigns a value.

== compares two values.

Integer division removes the decimal part.

% returns the remainder of a division.

Parentheses have the highest precedence.

Multiplication, division, and modulus are evaluated before addition and subtraction.

Best Practices

Use parentheses to make expressions easier to read.

Choose meaningful variable names.

Predict the output before running the program.
