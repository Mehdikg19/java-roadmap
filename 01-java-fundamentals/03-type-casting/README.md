Module 03 — Type Casting

Overview

This module introduces Type Casting in Java and explains how values are converted between different primitive data types.

The module covers both automatic and manual conversions, data loss, integer division, character-to-integer conversion, overflow behavior, and the importance of performing casting before or after an operation.

Topics Covered

Implicit (Widening) Casting

Explicit (Narrowing) Casting

Data Loss

Character to Integer Conversion

Integer Division

Casting Before vs After Operations

Overflow with byte

Learning Objectives

After completing this module, you should be able to:

Understand when Java performs automatic casting.

Perform explicit casting safely.

Predict data loss during narrowing conversions.

Explain why integer division behaves differently.

Understand overflow when converting to smaller data types.

Decide where casting should be applied in mathematical expressions.
