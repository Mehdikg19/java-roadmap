## Objective

Learn how to control the flow of a Java program using the `if`, `else if`, and `else` statements. Understand how conditional execution works and when each statement should be used.

---

## Prerequisites

- Variables
- Data Types
- Comparison Operators
- Logical Operators
- Boolean Values

---

## Learning Outcomes

After completing this topic, you will be able to:

- Write simple `if` statements.
- Use `if-else` for decision making.
- Create multiple conditions with `else if`.
- Understand nested `if` statements.
- Predict the execution flow of conditional statements.
- Choose the appropriate conditional structure for different problems.

## Topic Summary

The `if-else` statement is one of the fundamental control flow structures in Java. It allows a program to execute different blocks of code depending on whether a condition evaluates to `true` or `false`. Mastering conditional statements is essential before learning loops and more advanced programming concepts.
