# Additional Practice Ideas

Try creating programs that:

• Check whether a number is positive, negative, or zero.

• Determine whether a student has passed an exam.

• Compare two numbers and print the larger one.

• Check whether a person is eligible to vote.

• Determine whether a number is even or odd.

• Display different messages based on user age.

• Practice nested if statements.

• Predict the output before running every program.

# Challenge

Create a program that:

- Stores a student's score.
- Prints "Excellent" for scores 90 and above.
- Prints "Good" for scores from 75 to 89.
- Prints "Pass" for scores from 50 to 74.
- Prints "Fail" for scores below 50.

Try solving it before moving to the next topic.
