# 03 - for Loop

## Overview

In this topic, you learned how to use the `for` loop in Java to repeat a block of code a specific number of times.

## Learning Objectives

After completing this topic, you can:

- Understand why loops are useful.
- Write a basic `for` loop.
- Explain the three parts of a `for` loop.
- Count upward and downward.
- Change the loop step using `+=`.
- Print even and odd numbers.
- Read and write simple loop-based programs.
