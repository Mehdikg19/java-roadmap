# Additional Practice Ideas

Try creating programs that:

- Print the name of a weekday using a number.
- Print the name of a month using a number.
- Display different user roles.
- Build a simple calculator using switch.
- Print different menu options.
- Display different browser names.
- Practice switch with String values.
- Observe the effect of removing the break statement.
- Experiment with the default case.

---

# Challenge

Create a program that receives a number between **1** and **7** and:

- Prints the corresponding weekday.
- Prints **"Weekend"** for Saturday and Sunday.
- Prints **"Weekday"** for Monday to Friday.
- Prints **"Invalid Day"** for any other value.

Use:

- switch
- case
- break
- default
