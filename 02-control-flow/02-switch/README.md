## Important Notes

- `switch` compares the value of a single expression.
- Each `case` represents a possible matching value.
- The `break` statement exits the `switch` block.
- Omitting `break` causes execution to continue to the next `case` (fall-through).
- The `default` block executes when no `case` matches.
- `switch` supports several data types, including `int` and `String`.
